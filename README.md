# ljgabc.github.io

Personal GitHub Pages homepage for LJG.

## Local Preview

This is a static site with no build step. Open `index.html` directly in a browser, or run:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Publish

Push changes to the `main` branch of `ljgabc/ljgabc.github.io`. GitHub Pages serves user sites from that repository automatically.
